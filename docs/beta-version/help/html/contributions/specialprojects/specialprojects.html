iSphere Special Projects

There are some projects that are not intuitively to understand and that may need some additional explanation.

iSphere Compare Filters Compile-Dummy (RDi)

This project is a dummy project that is required to compile the iSphere Compare Filters Plugin (RDi) from RDi 8.0. It provides the org.eclipse.compare.ICompareFilter interface and the org.eclipse.compare.compareFilters extension point, which have been added with RDi 9.5 (Eclipse 4.4) and that are not available in RDi 8.0.

iSphere Security Plugin (WDSCi)

This plug-in emulates the Eclipse secure storage, which is not available for WDSCi 7.0.
