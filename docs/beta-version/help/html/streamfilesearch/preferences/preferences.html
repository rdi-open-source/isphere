Preferences

The available stream file search options are:

Batch resolve enabled - Specifies whether the selected stream file filters are resolved before the search dialog is presented or in a batch process after the search dialog has been closed for starting the search.
Maximum depth - Specifies the maximum depth of subdirectories included in the search. Use special value -1 for an unlimited depth. But be careful. IFS directory trees can grow very large.
This value cannot be changed on the iSphere search dialog, if batch mode is disabled, because the search items have already been resolved, at the time the dialog is displayed. Set Batch resolve enabled to false, to change the value on the search dialog.
Double-click mode - Specifies whether the Lpex editor is started in browse or edit mode, when double-clicking an IFS stream file.
Save results to - Specifies the folder in which to save stream file search results.
Auto-save enabled - Specifies if the content of the iSphere Stream File Search view is saved automatically.
Auto-save file name - Specifies the file name where search results are saved.
