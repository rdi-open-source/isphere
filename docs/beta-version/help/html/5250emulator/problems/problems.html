Known Problems

Eclipse Help Key Mapping

When pressing F1 or Shift+F1 in a TN5250j session, the eclipse help view will automatically be opened. You can prevent that, by changing the TN5250j key bindings of F1 and Shift+F1 to Alt+F1 and Alt+Shift+F1.

There is no known solution for disabling the Eclipse help key.

This problem has been fixed with iSphere 2.9.3.
