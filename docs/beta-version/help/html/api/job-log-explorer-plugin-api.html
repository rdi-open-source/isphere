Job Log Explorer Plug-in API

The iSphere Job Log Explorer plug-in API is provided by class: 

biz.isphere.joblogexplorer.externalapi.Access

The following views and editors are exposed by the API:

  • Job Log Explorer

There are methods for loading job logs from:

  • An active job
  • A QPJOBLOG spooled file
  • A QPJOBLOG spooled file, saved as a text file from ACS or iSphere
