Job Trace Explorer Plug-in API

The iSphere Job Trace Explorer plug-in API is provided by class: 

biz.isphere.jobtraceexplorer.core.externalapi.Access

The following views and editors are exposed by the API:

  • Job Trace Explorer

There are methods for loading job traces from:

  • A job trace session created with the STRTRC and ENDTRC commands
  • Json files (exported from the Job Trace Explorer view)
