Core Plug-in API

The iSphere Core plug-in API is provided by classes and interfaces: 

biz.isphere.core.externalapi.Access
biz.isphere.core.externalapi.IMessageFileCompareEditorConfiguration

The following views and editors are exposed by the API:

  • Binding Directory Editor
  • Data Area Editor
  • User Space Editor
  • Message File Editor
  • Message File Compare Editor

There is a method for each object type to load a given object into the editor.
