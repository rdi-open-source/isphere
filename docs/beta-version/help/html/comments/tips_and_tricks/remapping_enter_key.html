Remapping Enter Key

Usually mapping an action is no big deal on the User Key Actions preferences page of the LPEX Editor. But for the Enter key things are a bit different.

If you want to override the default action of the Enter key (enterRPG), you first have to turn-off the RPG-specific Enter key behavior for RPG/400 and/or ILE RPG.
