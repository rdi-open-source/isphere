Uninstall Instructions RDi

Follow these steps to uninstall the iSphere plug-in from RDi:

  • Start RDi 9.5.1.3+. (On Windows Vista, Windows 7 or Windows 8 RDi has to be started with "Run as administrator").
  • Select menu item "Help/Install New Software".
  • Click "What is already installed".
  • Tick all entries "iSphere" and click "Uninstall". The window "Uninstall Details" appears.
  • Click "Finish".
  • The window "Software Updates" with the message "You will need to restart ..." appears. Click "Restart Now".
  • After restart the window "Problems" can appear with the message "Could not restore workbench layout". You can ignore the message. Click OK. For future starts of the IDE, the message will not occur again.
