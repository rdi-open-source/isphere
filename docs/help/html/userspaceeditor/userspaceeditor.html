User Space Editor

The User Space Editor is used to edit user spaces.

This tool works well for simple user spaces or ad hoc changes.

If you have a more complex user space with multiple fields, consider the Data Space Editor, where you can define a layout for the user space using the iSphere Data Space Editor Designer.

To edit a user space:

  • Select a user space in the RSE tree
  • In the context menu (right click) select "iSphere User Space Editor"

In case you defined a data space editor, a popup dialog for selecting the editor will be displayed. Press the [Cancel] button to use the default user space editor:

After you have made your edits, close the editor and you will be prompted to save the changes:

You can use the standard Eclipse search to search for hexadecimal or string values:

  • x'E8FA32'

Note: Today options 'Whole word' and 'Selected lines' are not supported.
