5250 Emulator

With TN5250j for RDi 9.5.1.3+ you can run a 5250 Emulation in an own view/editor (TN5250j Sessions) natively in the IDE. You must no longer leave the IDE, if you need a 5250 session. (See section Session)

You can start SDA/RLU in an own view/editor (TN5250j Designer) within RDi 9.5.1.3+ for a source physical file member with type DSPF, MNUDDS or PRTF in view "Remote Systems" just by selecting the specific menu item. In order to use this feature the system value QRMTSIGN has to be set to *VERIFY or *SAMEPRF. (See section Designer Session)

With an user defined option in PDM "Work with member - WRKMBRPDM" you can start the LPEX-Editor, Screen Designer or Report Designer. (See section Start Eclipse Editor).

Furthermore you can start the Eclipse Compare/Merge Editor for comparing/merging source physical file members. (See section Start Eclipse Compare/Merge Editor).

The iSphere 5250 emulator is based on TN5250j at SourceForge at http://TN5250j.sourceforge.net/.
