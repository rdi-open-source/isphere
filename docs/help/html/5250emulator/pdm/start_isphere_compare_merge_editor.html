Start iSphere Compare/Merge Editor

With the iSphere 5250 emulator you can start an Eclipse the iSphere Compare/Merge editor from PDM with a user-defined option.

PDM User option

Start PDM (Command STRPDM) and select option 9 to work with the user-defined options. Then press F6 to add a new option. The following example uses "EC" (Eclipse Compare/Merge) to start an Eclipse editor:

OptionCommand
ECCALL PGM(ISPHERE/HANDLER) PARM('COMPARE' '&L' '&F' '&N')

Now you can use EC next to a source member to start the iSphere Compare/Merge editor from an iSphere 5250 session for the selected member.
