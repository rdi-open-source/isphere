Preferences

The available message file search options are:

Double-click mode - Specifies whether the iSphere message description editor is started in browse or edit mode, when double-clicking a message description.
Save results to - Specifies the folder in which to save message file search results.
Auto-save enabled - Specifies if the content of the iSphere Message File Search view is saved automatically.
Auto-save file name - Specifies the file name where search results are saved.
