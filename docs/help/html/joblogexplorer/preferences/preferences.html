Preferences

The available options of group Enable coloring are:
Enable coloring - This option will enable or disable coloring messages by severity at all.
Severity BLANK - This option controls the color of messages with a BLANK severity.
Severity 10 - This option controls the color of messages with a severity between 0 and 9.
Severity 20 - This option controls the color of messages with a severity between 10 and 19.
Severity 30 - This option controls the color of messages with a severity between 20 and 29.
Severity 40 - This option controls the color of messages with a severity between 30 and 39.
Severity 40 - This option controls the color of messages with a severity between 40 and 99.

The available options of group Parser settings are:
Date format - Specifies the date format that is used for parsing a job log spooled file. This option must be properly set when you want to use the iSphere Job Log Explorer for analyzing job log spooled files.
