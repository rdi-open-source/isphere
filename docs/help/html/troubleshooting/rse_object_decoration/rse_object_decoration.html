RSE Object Decoration is Missing

The RSE object decorator was introduced with iSphere v2.2.0. For details refer to the Host Object Decorator topic.

Due to enhancements introduced with iSphere v2.9.0, the object decorator had to be re-enabled after updating the iSphere plug-in.

In case that the objects displayed in the Remote Systems view are not decorated, follow the instructions given on the Label Decorations preferences page.
