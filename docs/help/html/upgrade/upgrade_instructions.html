Upgrade Instructions

Your preferences will be moved automatically to the new version.

Upgrading, from v2.x.x or later

If you are currently at iSphere v2.x.x or later, simply follow the Install Instructions. iSphere will automatically remove the plug-ins of the prior version.

Upgrading, from v1.x.x

If you are currently at iSphere v1.x.x, you must first manually remove the plug-ins for the prior version. Follow a link below for the manual uninstall instructions:

    Uninstall WDSC
    Uninstall RDP or RDi

After you have uninstalled the prior version plug-ins, follow the Install Instructions.

(Starting with v2.0.0 the iSphere project is open source and this required changing the plug-in IDs from de.taskforce to biz.isphere. Therefore, for this one upgrade, you must manually remove the plug-ins for the prior version. The uninstall instructions can also be found on the iSphere Wiki pages.)
