Journal Explorer Plug-in API

The iSphere Core plug-in API is provided by classes and interfaces: 

biz.isphere.journalexplorer.core.externalapi.Access
biz.isphere.journalexplorer.core.externalapi.IJournaledObject
biz.isphere.journalexplorer.core.externalapi.ISelectionCriteria

The following views and editors are exposed by the API:

  • Journal Explorer

There are methods for loading journal entries from:

  • Journals
  • Json files (exported from the Journal Explorer view)
  • Output files created with command DSPJRN

Methods starting with open directly open the specified object, whereas methods starting with explore displays a dialog for selecting the object to be opened.
